# College Decision Analytics Dashboard

## Overview
The College Decision Analytics Dashboard is an interactive Excel-based decision-support tool designed to help users evaluate and compare U.S. universities using publicly available higher education datasets. The dashboard integrates data from the U.S. Department of Education College Scorecard and the National Center for Education Statistics (NCES) to provide insights into institutional performance and outcomes.

The goal of this project was to transform large public datasets into a structured and user-friendly analytics tool that allows users to explore universities based on key academic and financial metrics.

## Features
- Interactive Excel dashboard for university comparison
- User inputs for **tuition budget** and **preferred state**
- Dynamic ranking returning the **top 5 matching universities**
- Comparison of key institutional metrics:
  - Tuition cost
  - Admission rate
  - Average SAT score
  - Median earnings after graduation
- API-based school logo integration
- Interactive campus location mapping
- Summary of the **top majors offered at each university**

## Data Sources
- U.S. Department of Education – College Scorecard  
- National Center for Education Statistics (NCES)

## Technologies Used
- Microsoft Excel
- Power Query (ETL and data integration)
- Dynamic Excel formulas
- Data visualization and dashboard design

## How to Use
1. Download the Excel workbook from the repository.
2. Open the file in Microsoft Excel.
3. Navigate to the **Info/Input sheet**.
4. Enter your desired **tuition budget** and **preferred state**.
5. The dashboard will automatically return the **top 5 matching universities** and display detailed comparisons.

## Project Goal
This project demonstrates how public datasets can be transformed using ETL processes and data modeling techniques to build an interactive analytics dashboard that supports data-driven decision making.
